# Contributing

Thank you for considering contributing to **septa-mcp**. To keep the project healthy and easy to maintain, please follow these guidelines:

## Setup
1. Create and activate a Python 3.11+ virtual environment.
2. Install dependencies:
   ```bash
   pip install -e .[dev]
   ```

## Development
- Run `ruff` to lint the codebase.
- Execute `pytest` to ensure all tests pass.
- Format code using Ruff's formatter where possible.

## Pull Requests
- Keep changes focused and describe the motivation in the PR description.
- Add tests for new functionality when applicable.
- Ensure CI checks pass before requesting review.
